### HTTP Request
POST `/v5/bybitpay/settlement_record`

:::warning Feature Currently Unavailable
The Settlement Report API is temporarily not supported.

If you need access to settlement data, please contact us:
- Email: [bybitpay@bybit.com](mailto:bybitpay@bybit.com)
- Business inquiries: [Partner with us](https://forms.gle/Kou91v4MUT61C8o97)
:::

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|merchantId |**true** |string |Merchant ID|
|clientId |**true**|string |Merchant client ID|
|[paymentType](data-model/payment-type) |**true**|string |Payment Type|
|settleAccountBeginDate |**true**|string |Settle account date, like `2025-09-10`|
|settleAccountEndDate |**true**|string |Settle account date, like `2025-09-12`|
|start |**true**|int |Start, default 0|
|limit |**true**|int |Limit, default 100|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|list |array&lt;[SettlementOrderType](data-model/settlement-order)&gt; |Settlement order list|

### Request Example

#### Query Settlement Records by Date Range
```http
POST /v5/bybitpay/settlement_record HTTP/1.1
Host: api.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: xxxxxxxxxxxxxxxxxx
X-BAPI-TIMESTAMP: 1736233200000
X-BAPI-RECV-WINDOW: 5000
Content-Type: application/json

{
    "merchantId": "305142568",
    "clientId": "client_001",
    "paymentType": "E_COMMERCE",
    "settleAccountBeginDate": "2026-01-01",
    "settleAccountEndDate": "2026-01-07",
    "start": 0,
    "limit": 100
}
```
